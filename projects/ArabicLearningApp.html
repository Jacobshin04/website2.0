Arabic Learning App

  • Tech Stack: React, Next.js, Node.js, Express.js, MongoDB, TypeScript, JavaScript, Tailwind CSS
  • Github URL: Project Link

Project Description

Built with JavaScript, HTML, and CSS for an interactive UI, this Arabic verb learning app is designed to help Korean learners grasp the 60 verb patterns, eight personal pronouns, and three tenses in Arabic.

The platform allows users to input over 130 verbs, automatically generating color-coded verb tables that highlight stems and endings for better visualization. By making conjugation patterns more intuitive, the app enhances the learning process and engagement.
